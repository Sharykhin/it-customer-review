Customer Review Email System:
=============================
A simple test task implementation from iTechArt.

Requirements:
-------------

[Docker](https://www.docker.com/)

Local development:
-----------------

1. Build images:
```bash
docker-compose build
```

2. Run containers:
```bash
docker-compose up -d
```

3. Ensure all containers were run successfully:
```bash
docker-compose ps
```
